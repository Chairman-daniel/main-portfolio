Personal Portfolio Website <br>
Overview <br>
This is a responsive and professional personal portfolio website showcasing my skills, projects, and experience. It serves as a platform for potential clients, employers, or collaborators to learn about my work and contact me directly. The design prioritizes user experience, aesthetics, and accessibility, with options for both light and dark themes. <br>

Features <br>
About Me Section: A brief introduction outlining my background, skills, and expertise. <br>
Projects Section: Highlights some of my best projects with links to live demos or repositories. <br>
Contact Section: A functional contact form for visitors to reach out directly. <br>
Responsive Design: Optimized for various devices, ensuring seamless experience on desktops, tablets, and mobile devices. <br>

Technologies Used <br>
Frontend: HTML, CSS, JavaScript <br>
Backend: PHP (for handling contact forms and form submissions) <br>
Database: MySQL (optional, for storing form submissions) <br>
Version Control: Git
